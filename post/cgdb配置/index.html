1.安装

git clone git://github.com/cgdb/cgdb.git
cd cgdb
./autogen.sh
./configure --prefix=/usr/local
make
sudo make install

# 报错解决方法
sudo apt-get install automake libncurses5-dev flex texinfo libreadline-dev

2.配置

vim ~/.cgdb/cgdbrc

set ignorecaseset
ts=4
set wso=vertical
set eld=shortarrow
set hls
map <F9> :until<cr>

3.快捷键

F5 - Send a run command to GDB. F6 - Send a continue command to GDB. F7 - Send a finish command to GDB. F8 - Send a next command to GDB. F10 - Send a step command to GDB.
